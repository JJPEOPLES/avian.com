System Requirements

Before installing Avian OS, make sure your system meets the following requirements:

Component Minimum Recommended
Processor 1 GHz dual-core 2 GHz quad-core or better
Memory (RAM) 2 GB 4 GB or more
Storage 20 GB 40 GB or more
Graphics VGA capable of 1024x768 Graphics card with 3D acceleration
Boot UEFI or Legacy BIOS UEFI with Secure Boot support

Preparation

1. Download Avian OS

First, download the Avian OS ISO file from our downloads page.

2. Create a Bootable USB Drive

You'll need to create a bootable USB drive to install Avian OS. Here's how to do it on different operating systems:

On Windows, we recommend using Rufus or Etcher:

  1. Download Rufus or Etcher
  2. Insert your USB drive (at least 4GB)
  3. Open Rufus or Etcher
  4. Select the Avian OS ISO file
  5. Select your USB drive
  6. Click "Start" or "Flash" and wait for the process to complete

On macOS, we recommend using Etcher:

  1. Download Etcher
  2. Insert your USB drive (at least 4GB)
  3. Open Etcher
  4. Select the Avian OS ISO file
  5. Select your USB drive
  6. Click "Flash" and wait for the process to complete

On Linux, you can use Etcher or the dd command:

  1. Using Etcher:
    • Download Etcher
    • Insert your USB drive (at least 4GB)
    • Open Etcher
    • Select the Avian OS ISO file
    • Select your USB drive
    • Click "Flash" and wait for the process to complete
  2. Using dd command:
    • Insert your USB drive (at least 4GB)
    • Open a terminal
    • Find your USB drive with lsblk (e.g., /dev/sdb)
    • Run: sudo dd bs=4M if=path/to/avian-os.iso of=/dev/sdX status=progress oflag=sync (replace /dev/sdX with your USB drive)

3. Backup Your Data

Before installing any operating system, it's always a good idea to back up your important data.

4. Prepare Your System

If you plan to dual-boot with another operating system, make sure you have free space on your hard drive or create a new partition.

Installation Steps

1. Boot from USB

  1. Insert the bootable USB drive into your computer
  2. Restart your computer
  3. Enter your computer's boot menu (usually by pressing F12, F2, or Esc during startup)
  4. Select your USB drive from the boot menu

2. Start the Installation

  1. Once Avian OS boots, you'll see the welcome screen
  2. Select "Install Avian OS"
  3. Choose your language, time zone, and keyboard layout

3. Disk Partitioning

You'll be presented with several installation options:

  • Install alongside existing OS: This will set up a dual-boot system
  • Erase disk and install: This will erase everything on your disk and install Avian OS
  • Something else: This allows you to manually configure partitions

Note: If you choose "Erase disk and install", all data on the selected disk will be permanently deleted.

4. Create User Account

  1. Enter your name
  2. Choose a computer name
  3. Create a username
  4. Set a strong password
  5. Choose whether to log in automatically or require a password

5. Installation Process

  1. Review your choices and click "Install"
  2. The installation will begin and may take 10-20 minutes depending on your hardware
  3. Once complete, you'll be prompted to restart your computer

Post-Installation

1. First Boot

After restarting, you'll be greeted with the login screen. Enter the username and password you created during installation.

2. Update Your System

It's important to update your system to get the latest security patches and software updates:

sudo apt update
sudo apt upgrade

3. Install Additional Software

Avian OS comes with essential software pre-installed, but you can install additional software from the Software Center or using the command line.

4. Configure System Settings

Take some time to configure your system settings according to your preferences:

  • Appearance and themes
  • Display settings
  • Power management
  • Network connections
  • User accounts

Troubleshooting

Boot Issues

If you're having trouble booting from the USB drive:

  • Make sure your computer is set to boot from USB in the BIOS/UEFI settings
  • Try disabling Secure Boot in your BIOS/UEFI settings
  • Try recreating the bootable USB drive with a different tool

Display Issues

If you experience display problems during or after installation:

  • Try booting with the "Safe Graphics" option from the boot menu
  • Add the nomodeset parameter to the kernel boot options

Network Issues

If you're having trouble connecting to the internet:

  • Check your network hardware is properly connected
  • Try using a wired connection if possible
  • Some wireless adapters may require additional drivers

Getting Help

If you encounter any issues not covered here, please visit our Community page for support.
